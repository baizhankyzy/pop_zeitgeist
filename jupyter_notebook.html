Jupyter notebooks

The first notebook is to scrape the data from the web and save them locally. The scraper may take around 4 hours to finish working. Therefore, please only check the codes for reference and ignore the errors returned.

In the second notebook we are going to manipulate, analyze and visualize the metadata of the music. The error returned in a few blocks is caused by the fact that the Mercury web app (the embedding tool we are currently using) can not import one of the Python libraries we used.

The third notebook is to clean, analyze and visualize textual data of lyrics. During this process, saving cleaned files to the local path and reading locally is inevitable, which makes the complete embedding using a web app impossible currently. We are sorry for the inconvenience, please view this notebook at the GitHub repository: https://github.com/delete4ever/pop_zeitgeist/blob/main/3_text_analysis.ipynb
